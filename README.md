# Pong-Game
As my first C++ Game, I chose the classic Pong Game as a way to solidify my recent studies in C++ and SFML.

## Controls:
- Player 01: `W | S`
- Player 02: `Up | Down`
- Increase Ball Speed: `Z`
- Decrease Ball Speed: `X`
